# Trips UI

A simple project to manage trip requests.

[Demo](http://safire-trips-ui.surge.sh/)

### Requirements
Node >= 6.0.0 and NPM => 3.0.0

### Run

To run project, first install dependencies:

```sh
npm install
```
```sh
bower install
```
Before that, you can serve at localhost:3000:

```sh
gulp serve
```

### Tests

To execute tests:

```sh
gulp test
```
